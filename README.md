📖 English Janala (ইংরেজি জানালা)
সহজেই ইংরেজি শব্দ শেখার একটি আধুনিক প্ল্যাটফর্ম।

English Janala একটি ইউজার-ফ্রেন্ডলি ওয়েবসাইট যেখানে আপনি খেলাচ্ছলে নতুন নতুন ইংরেজি শব্দ এবং তাদের সঠিক অর্থ শিখতে পারবেন। এটি বিশেষ করে তাদের জন্য তৈরি করা হয়েছে যারা প্রতিদিনের ভোকাবুলারি বাড়াতে চান।

✨ ওয়েবসাইটের প্রধান সুবিধাসমূহ:
সহজ লেসন সিস্টেম: ওয়েবসাইটটি বিভিন্ন লেসনে ভাগ করা, ফলে আপনি ধাপে ধাপে শিখতে পারবেন।

সঠিক উচ্চারণ: প্রতিটি শব্দের সাথে একটি 'সাউন্ড' বাটন আছে, যা আপনাকে শব্দটির সঠিক ইংরেজি উচ্চারণ শুনতে সাহায্য করবে।

বিস্তারিত তথ্য: 'Details' বাটনে ক্লিক করে আপনি শব্দের অর্থ এবং সেটি বাক্যে কীভাবে ব্যবহার করতে হয় তা দেখতে পারবেন।

পছন্দের শব্দ সেভ: আপনার পছন্দের বা কঠিন শব্দগুলো 'হার্ট' আইকনে ক্লিক করে আলাদাভাবে সেভ করে রাখার সুবিধা।

রিয়েল-টাইম সার্চ: যেকোনো নির্দিষ্ট শব্দ খুব দ্রুত খুঁজে পাওয়ার জন্য এতে একটি পাওয়ারফুল সার্চ বক্স আছে।

স্পেশাল ডিজাইন: লেসন ৪ এবং ৭-এর জন্য রয়েছে বিশেষ আকর্ষণীয় ডিজাইন, যা আপনার শেখার আগ্রহ বাড়িয়ে দেবে।

সব ডিভাইসে চলে: আপনি আপনার মোবাইল, ট্যাবলেট বা পিসি—যেকোনো কিছু দিয়ে এটি স্বাচ্ছন্দ্যে ব্যবহার করতে পারবেন।
